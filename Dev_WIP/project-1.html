p5js Project 1

Pre-trained models are used in real-world applications to make decisions about various aspects of our lives. I also use pre-trained models in this project to show how they are limited by the data they are trained on. Yet, they are used to influence important aspects of our lives. Pre-trained model has learnt based on a fixed dataset. If i show it an image not from it’s dataset, it will not know about it.

This example focuses on the usage of “pre-trained models” in Machine Learning using image classification. The user can show an object to the webcam which will recognise the object. Sometimes, the recognition is not accurate. The user can search through the list of images the model is trained on, to see if the object they are showing is a part of it.
